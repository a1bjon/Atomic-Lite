               -Atomic Lite Editor Help-

[Open File] - Opens a file explorer and will open any text based file.

[Save As] - Opens a file explorer to save current text environment to specified destination(specify name and file extension e.g. 'test.txt').

[RM File] - Removes currently opened file.

[▲ size ▼] - Increases or decreases text font size in environment based on value.

[Font] - Changes environment font to selected font.

[Theme] - Changes text editors theme to selected theme.

[Copy Text] - Copies all test in environment to clipboard.

[Reset] - Resets all settings applied to text editor to default settings.

[Quit] - Closes text editor
